应用 快捷键

win 快捷键

WIN键 + [A]: 快速设置面板
WIN键 + [B]: 快速跳转系统托盘
WIN键 + [C]: Microsoft Teams
WIN键 + [D]: 快速显示桌面
WIN键 + [E]: 打开资源管理器
WIN键 + [F]: 反馈中心
WIN键 + [G]: game bar
WIN键 + [H]: 语音听写
WIN键 + [I]: 系统设置
WIN键 + [K]: 投屏
WIN键 + [L]: 锁屏
WIN键 + [M]: 窗口最小化
WIN键 + [N]: 通知面板
WIN键 + [P]: 投影
WIN键 + [Q]: 一键搜索
WIN键 + [R]: 运行
WIN键 + [S]: 一键搜索
WIN键 + shift + [S]: 截图
WIN键 + [T]: 已打开程序缩略图
WIN键 + [U]: 辅助功能
WIN键 + [V]: 云剪贴板
WIN键 + [W]: 小组件
WIN键 + [X]: 开始菜单
WIN键 + [Z]: 窗口布局
WIN键 + [PrintScreen]: 截屏

linux 快捷键

Ctrl + r ：搜索历史命令
Ctrl + s ：暂停终端。（suspend）
Ctrl + q ：退出当前状态，如暂停状态。（quit）
Ctrl + d ：退出当前终端。
Ctrl + z ：暂停在终端运行的任务，配合 jobs、fg、bg 等命令使用。
Ctrl + c ：终止在终端中运行的任务。
